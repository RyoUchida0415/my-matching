サイト概要
マッチングアプリ

サイトテーマ
人柄がよくわかるマッチングアプリ。既存のマッチングアプリに簡略版のブログ機能を追加実装しその人の人柄をより知ることができようなもの。

テーマを選んだ理由
既存のマッチングアプリでは人柄をプロフィールでしか判断できず情報量が少ないと感じたから。

ターゲットユーザ
既存のマッチングアプリに会員の情報量で不満を持つ恋愛をしたい男女。
